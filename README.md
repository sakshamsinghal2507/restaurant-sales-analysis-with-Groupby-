# Restaurant Sales Analysis

## Project Overview
This project analyzes restaurant sales data to understand revenue patterns and customer behavior using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Sales per day using GroupBy
- Average spending by gender
- Tips analysis (Lunch vs Dinner)
- Correlation using Heatmap

## Key Insights
- Some days generate higher sales
- Dinner time generates more tips
- Higher bills lead to higher tips
- Strong relationship between total_bill and tip
